# 🎮 OmokDuelMasters

> Photon 기반 실시간 PvP 오목 전략 게임

---

## 📌 프로젝트 소개

**OmokDuelMasters**는 전통적인 오목 게임에 **온라인 실시간 대전 시스템**을 결합한 프로젝트입니다.
간단한 규칙 속에서도 전략성과 경쟁 요소를 극대화하여, 플레이어 간 긴장감 있는 PvP 경험을 제공합니다.

---

## 🎯 기획 개요

* **장르**: 턴제 전략 / PvP
* **플랫폼**: PC (Unity)
* **개발 인원**: 3인 팀 프로젝트
* **개발 기간**: (작성 필요)

### ✔ 핵심 컨셉

* 누구나 쉽게 즐길 수 있는 오목 게임
* 실시간 유저 간 경쟁 중심의 게임 플레이
* 빠른 매칭과 간결한 게임 진행 구조

## 🧠 주요 기능

### 🎮 게임 플레이

* 오목 규칙 기반 턴제 진행
* 5목 완성 시 자동 승리 판정
* 턴 상태 동기화 처리

### 🌐 멀티플레이

* Photon 기반 실시간 네트워크

* 플레이어 매칭 기능

### 📦 시스템

* 인벤토리 및 확장 가능한 구조 설계
* UI 인터랙션 처리

## 💡 담당 기능 (기여도)
